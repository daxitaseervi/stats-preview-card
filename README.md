## Overview
This project is a responsive stats preview card component built with HTML and CSS. The goal was to recreate the provided design as accurately as possible while practicing responsive layouts, semantic HTML, and modern CSS techniques.

### Key learnings
- Built a responsive layout using Flexbox.
- Improved my understanding of CSS positioning and layering.
- Practiced creating image overlays with pseudo-elements.
- Learned how to structure reusable CSS using the BEM naming convention.
- Gained more experience with responsive design using media queries.
- Improved my Git workflow by making regular commits and deploying the project with GitHub Pages.
